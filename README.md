
# Simple Blog App

a simple web blog application built using Laravel 12 + TailwindCSS 4.0
## Authors

- [@mrizkyprtm](https://www.github.com/mrizkyprtm)


## Features

- Authentication (Login, Register)
- Authorization
- Edit Profile User
- CRUD Posts
- Comment on Post
- Reply Comment
- Delete Comment
- Search Posts
- Filter Posts by Category
- Sort Post by Date
